# APW-TECH Web

Next.js implementation of the supplied APW-TECH client website, migrated from the provided standalone HTML source with matching structure, styling, bilingual content, and interactions.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`

## Notes

- The contact form intentionally mirrors the original HTML behavior and shows a client-side success state only.
- The repeating background image was extracted into `public/bg.png` to keep styles manageable for deployment.
